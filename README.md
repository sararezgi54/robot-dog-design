# 🤖 Robot Dog Design

## Overview
This repository contains the 3D mechanical design of a quadruped robot (Robot Dog) created using Tinkercad.

The project focuses on designing the robot's body and legs while considering joint placement, stability, and overall mechanical structure.

## Degrees of Freedom (DOF)

- Each leg has 2 joints.
- Total Degrees of Freedom: 8 DOF.

---

## Motor Selection

- Proposed Motor: SG90 Servo Motor.

---

## Stability

The robot is designed with four legs placed symmetrically to improve balance and stability.

---

## Proposed Movement

The robot is intended to walk using alternating leg movements.

---

## Expected Mechanical Issues

- Joint stress during movement.
- Limited motor torque.
- Balance may be affected on uneven surfaces.
---
## Project Objectives
- Design a simple four-legged robot.
- Create a stable mechanical structure.
- Include leg joints for movement.
- Prepare the design for future servo motor integration.
